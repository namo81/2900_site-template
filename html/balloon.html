말풍선

클릭형

스타일은 css 에서 별도 설정

기본적으로는 말풍선이 버튼 아래 및 버튼 좌측 기준에 맞추어 생성되며, data-top / data-left 를 통하여 위치 조정 가능(px단위)

말풍선 내용이 들어가는 영역입니다. 
<div class="balloon-wrap">
	<button type="button" class="btn-balloon" data-top="5" data-left="-5"><span>버튼</span></button>
	<div class="cnt-balloon">
		<div class="cnt">
			말풍선 내용이 들어가는 영역입니다.
		</div>
		<button type="button" class="btn-ball-close">말풍선 닫기</button>
	</div>
</div>
